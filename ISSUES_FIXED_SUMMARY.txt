   Result: Safely redirects to HomeScreen if no restaurant selected

   Solution: Added guard clause at component start

✅ 4. MENU NOT FILTERED BY RESTAURANT
   Problem: All menu items shown regardless of restaurant selection
   Solution: Added .filter((item) => item.restaurantId === restaurant.id)
   Result: Each restaurant shows only its own menu items

✅ 5. DELIVERY FEE HARDCODED
   Problem: CartScreen always showed 500 FCFA, should vary by restaurant
   Solution: Made deliveryFee a prop with default, pass from App.js
   Result: Delivery fee now matches selected restaurant

✅ 6. NO FORM VALIDATION
   Problem: Login/signup forms accept empty, invalid, mismatched inputs
   Solution: Added comprehensive validation:
     - Email regex validation
     - Phone number validation (8+ digits)
     - Password minimum 6 characters
     - Password confirmation matching
     - Empty input checks
     - Error message display with styling
   Result: Forms now properly validate all inputs
